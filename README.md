# Protected Repository

This repository has been encrypted to protect the intellectual property. 

## How to Decrypt
To access the source code, you must have the decryption password.

1. Ensure you have **Node.js** installed.
2. Run the following command in the root directory:
   ```bash
   node secure_file.js decrypt . <your_password>
   ```

## Files
- `secure_file.js`: Utility script for encryption/decryption.
- `*.enc`: Encrypted source files.
